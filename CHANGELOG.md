# Changelog

## [3.0.0] - 2026-03-30

### Added
- Complete 31-chapter trading playbook as single-page HTML app
- PWA support: manifest.json, service worker for offline access
- Mobile-first responsive design with swipe navigation
- Book-style page transitions with slide animations
- Progress bar showing reading position
- Table of contents with chapter links
- Structured 12-week curriculum across 3 phases:
  - Phase 1 (Weeks 1-4): Foundation — markets, candlesticks, chart patterns, S/R, Fibonacci, indicators, sessions
  - Phase 2 (Weeks 5-8): Strategy — risk management, multiple timeframes, strategy building, volume profile, psychology, SMC/ICT, Wyckoff, backtesting, macro analysis
  - Phase 3 (Weeks 9-12): Execution — live demo trading, advanced trade management, strategy refinement, correlation analysis, Go/No-Go evaluation
- Reference chapters: leverage table, free resources, backtesting methodology, formal trading plan template, Obsidian ledger setup, common traps
- Keyboard navigation (arrow keys)
- Touch swipe navigation for mobile
- GitHub Pages deployment as installable PWA

### Technical
- Zero dependencies — vanilla HTML/CSS/JS
- Google Fonts: DM Serif Display, JetBrains Mono, Outfit
- CSS custom properties for theming
- Dark theme with green (#00e676) accent
- Responsive typography with clamp()
- Custom scrollbar styling
- Proper meta tags for mobile web app
