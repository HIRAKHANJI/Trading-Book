# Changelog

All notable changes to The Trading Playbook are documented here.

## [1.0.0] - 2026-03-30

### Added
- Complete 30-page trading playbook covering a 12-week curriculum
- **Foundation**: Non-negotiable rules, platform setup guide (TradingView, Bybit Testnet, Naga Demo, SMRT Algo)
- **Phase 1 (Weeks 1-4)**: Markets & leverage, candlestick patterns, chart patterns, support/resistance, market structure, Fibonacci retracements, indicators, session analysis
- **Phase 2 (Weeks 5-8)**: Risk management, position sizing, strategy building, trading psychology, journaling, liquidity & smart money concepts (order blocks, FVGs, Judas Swing), Wyckoff method, backtesting, macro & fundamental analysis
- **Phase 3 (Weeks 9-12)**: Live demo execution, advanced trade management (trailing stops, scaling, partial TPs), strategy refinement, correlation analysis, go/no-go live trading evaluation
- **Reference pages**: Leverage reality table, free resources master list, backtesting methodology, formal trading plan template, Obsidian trading ledger setup
- Mobile-first dark theme UI with swipe navigation
- Table of contents with quick-jump links
- Progress bar tracking
- PWA support: web app manifest, service worker for offline caching
- GitHub Pages deployment via `index.html` entry point
- Project documentation: README.md, CLAUDE.md, ARCHITECTURE.md
