# The Trading Playbook

A comprehensive, research-backed trading education platform built as a Progressive Web App. 31 chapters covering leveraged crypto, forex, and multi-asset trading over a structured 12-week curriculum.

## Live Site

Hosted via GitHub Pages as a PWA — installable on mobile and desktop.

## What's Inside

- **31 chapters** organized across 3 phases (Foundation, Strategy, Execution)
- **12-week structured curriculum** with daily 45-minute sessions
- Covers: candlesticks, chart patterns, S/R, Fibonacci, indicators, risk management, SMC/ICT, Wyckoff, psychology, journaling, and more
- Position sizing formulas, leverage progression system, and Go/No-Go evaluation
- Obsidian trading ledger setup guide
- Free resources master list — zero paid groups required

## Tech Stack

- Single-file HTML/CSS/JS — no build step, no dependencies
- PWA with service worker for offline access
- Mobile-first responsive design with swipe navigation
- Book-style page transitions with progress bar
- Google Fonts: DM Serif Display, JetBrains Mono, Outfit

## Running Locally

Open `index.html` in any browser. That's it.

## PWA Installation

Visit the GitHub Pages URL on mobile, tap "Add to Home Screen." The app works offline after first load.
